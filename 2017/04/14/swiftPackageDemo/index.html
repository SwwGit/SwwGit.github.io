发表于 2017-04-14 | 分类于 swift | | 阅读次数

Swift Package Manager(SPM)是苹果官方的Swift包管理工具,可以用来管理 Swift 代码的分配，它为 Swift 编译系统集成了自动进行下载、编译和连接依赖的过程。

官方有一个对于SPM的示例教程，大家可以去看看:Package Manager

为什么呢？

Swift通过 Modules 来管理代码，默认情况下，所有在 Sources 目录下的文件都在同一个module中，我们在Sources中新建了一个文件夹，就是新增了一个module－Car，也就没有了默认的module了，所以main.swift被孤立在外边。
